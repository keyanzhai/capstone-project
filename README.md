```
source mp_env/bin/activate

```

## Output Format

<https://google.github.io/mediapipe/solutions/pose#pose_landmarks>

* pose_landmarks
  * x
  * y
  * z
  * visibility
* pose_world_landmarks
* segmentation_mask
